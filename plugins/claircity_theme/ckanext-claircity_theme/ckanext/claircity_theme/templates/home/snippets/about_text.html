This the data portal of the ClairCity project.

This is the development version, any data can be removed any time!

ClairCity is a research project, funded through the EU’s Horizon 2020 program, where a number of research institutes together with 6 European cities work together to improve air quality in these cities. For more information on the project we refer to the the ClairCity website.

The purpose of this data portal is to facilitate data management within the project between project partners. It can be used by ClairCity members to store and exchange research data and models or to link to other publicly available data sources that are used in the project. It is not allowed to upload any privacy-sensitive data.

This project has received funding from the European Union’s Horizon 2020 research and innovation programme under grant agreement No 689289.

Hosted by: Statistics Netherlands.
